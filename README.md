Solace

Little Things, Lasting Comfort.

Solace is a simple, aesthetic lifestyle website built to represent a brand focused on emotional comfort and mindful living. It combines soft design, handmade product ideas, and a small AI mood-based recommendation system to suggest comfort items based on how a user feels.

⸻

Features

* Clean homepage with brand identity
* AI Mood Engine that recommends products based on emotions
* Product showcase (candles, bracelets, bookmarks, perfumes)
* About section explaining the brand philosophy
* Team section with creators
* Soft, aesthetic UI with warm tones

⸻

How It Works

The AI Mood Engine asks the user how they feel (calm, stressed, happy, tired) and then suggests comfort-based products like candles, bookmarks, bracelets, and perfumes that match their emotional state.
